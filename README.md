# 📊 SuperStore Sales Dashboard (Microsoft Excel)

## Overview

This project is an interactive **Business Intelligence dashboard** built in **Microsoft Excel** to analyze SuperStore sales performance. It transforms raw sales data into meaningful insights using Excel's advanced analytical features.

## Features

* 📈 Executive KPI Cards
* 💰 Sales & Profit Analysis
* 🏙 Top & Bottom Performing Cities
* 📦 Top Product Categories & Sub-Categories
* 📊 Profit vs. Loss Analysis
* 🎯 Discount Impact Analysis
* 🔎 Interactive Slicers for Dynamic Filtering
* 🧭 Multi-page Dashboard Navigation

## Tools & Technologies

* Microsoft Excel
* Power Query
* Power Pivot
* DAX
* Pivot Tables
* Pivot Charts
* Slicers

## Key Insights

* Identified the highest and lowest performing cities.
* Analyzed profit trends across product categories.
* Evaluated the impact of discounts on profitability.
* Highlighted top-performing products based on sales and orders.
* Designed an interactive dashboard for data-driven decision-making.
